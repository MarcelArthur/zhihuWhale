# 简介
#### onewhale———仿知乎网站Demo 
# 开发语言和环境

- 开发工具：Idea

- 开发语言：Java8

- 数据库：Oracle 11g

- 后端：Servlet

- 页面渲染：JSP

- 前端：Bootstrap，JQuery

- 富文本编辑器：wangEditor

# 主要功能

- 登录/注册功能

- 提问功能

- 回答功能

- 评论功能

- 话题分类搜索功能

- 用户权限配置：
		用户分为游客、普通用户、管理员用户三种角色。管理员用户可以管理普通用户的登录及发帖权限。同时将用户的发言暂时屏蔽。
